## important first thing to do

To launch Chrome with the remote debugging port enabled so Playwright can connect to it, please follow these steps:

### Windows Launch Instructions

1. **Close All Running Chrome Windows:**
   Chrome must be completely closed first. If a normal Chrome instance is already running, it may block the debugging server from starting.

2. **Open the Windows "Run" Dialog:**
   Press **`Win + R`** on your keyboard.

3. **Paste the Launch Command:**
   Paste the following command into the Run box and press **Enter**:

   ```cmd
   chrome.exe --remote-debugging-port=9222 --user-data-dir="C:\ChromeDebugProfile"
   ```

   *(Note: This uses the standard `C:\ChromeDebugProfile` directory referenced in your `Project-workflow.md` to keep your signed-in Gemini/AI Studio accounts isolated and active.)*

4. **Verify It Works:**
   Once Chrome opens, you can verify it is listening by opening a tab inside it and navigating to:
   `http://127.0.0.1:9222/json/version`
   If you see a page with JSON text, the debugging server is successfully running.
