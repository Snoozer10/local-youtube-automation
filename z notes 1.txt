How to use this:
Make sure you have your URLs in youtube_urls.txt.
Ensure your .txt config files (voice_option_notes.txt, gemini_model.txt, etc.) are configured.
Open your terminal and simply type:
python run_agency.py


or  Launch (run.bat) (OLD)



Tip: If users run transcription on CPU, recommend they open voice_option_notes.txt and change Whisper Model: small to Whisper Model: base or Whisper Model: tiny to speed up CPU-bound processing [transcribe_audio.py].



Consolidate Setup Credentials
Ensure users install the correct external tools in their environment. Create a simple requirements.txt to install all dependencies cleanly.
Create a requirements.txt file in your root folder:

playwright>=1.40.0
youtube-transcript-api>=1.2.4
python-docx>=0.8.11
moviepy>=2.0.0
Pillow>=10.0.0
pypdf>=3.17.0
torch
openai-whisper


Users can install everything using:
pip install -r requirements.txt
playwright install
